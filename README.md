## 🚀 Deployment

### GitHub Pages (Free)

1. Create a new repository on GitHub (e.g. `eyasu-portfolio`)
2. Upload `eyasu_misganu_freelancer.html` and rename it to `index.html`
3. Upload this `README.md`
4. Go to **Settings → Pages**
5. Under **Source**, select `main` branch and `/ (root)`
6. Click **Save** — your site will be live at:
   ```
   https://yourusername.github.io/eyasu-portfolio/
   ```

### Netlify (Free, Drag & Drop)

1. Go to [netlify.com](https://netlify.com) and sign up
2. Rename the HTML file to `index.html`
3. Drag and drop the file onto the Netlify dashboard
4. Your site is live instantly with a free URL

### Vercel (Free)

1. Go to [vercel.com](https://vercel.com) and sign up
2. Connect your GitHub repository
3. Vercel auto-detects and deploys — done!
